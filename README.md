# 🧠 DSA Smart Hint Assistant - Chrome Extension

A Chrome extension that provides personalized, approach-specific hints for DSA problems on platforms like LeetCode.

---

## 🔍 Purpose

- Default platform hints are often generic and not tailored to user-written solutions.
- One problem can be solved in multiple valid ways; this extension supports solution-specific guidance.
- Beginners often need help with syntax, structuring, and logic beyond generic advice.

---

## 💡 Features

- Approach-aware hints based on the user's current solution.
- Debug support to understand wrong outputs without changing the original logic.
- Dry run simulation to trace code execution.
- Progress tracking and submission history overview.
- Syntax assistance for beginners in supported languages.

---

## ⚙️ Tech Stack

- **Frontend:** React, Tailwind, Vite
- **Backend:** Node.js, Express, OpenAI API, mongoDB
- **Model/LLM:** Qwen 2.5 (by Alibaba) via OpenRouter
- **Hosting:** Render
